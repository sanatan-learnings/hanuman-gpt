## About Hanuman Kavacham

Kavacham literally means "armor" in Sanskrit. This prayer invokes Hanuman's protection for different parts of the body and aspects of life, creating a spiritual shield around the devotee.

## Structure

Each verse file should follow the standard format:
- `verse_01.md`, `verse_02.md`, etc.
- Contains Sanskrit/Devanagari text, transliteration, word meanings
- Specifies which part of the body or life aspect is being protected
