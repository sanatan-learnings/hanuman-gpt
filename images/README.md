# Images Directory

This directory contains visual representations for each verse of the Hanuman Chalisa.

## File Naming Convention

- Opening dohas: `doha_01.jpg`, `doha_02.jpg`
- Main verses: `verse_01.jpg`, `verse_02.jpg`, ... `verse_40.jpg`
- Closing doha: `doha_closing.jpg`

## Image Requirements

- Format: JPG or PNG
- Recommended size: 800x600px or similar aspect ratio
- Content: Iconographic depictions, traditional artistic interpretations, or symbolic elements related to each verse

## Total Files Needed

43 image files (2 opening dohas + 40 verses + 1 closing doha)
